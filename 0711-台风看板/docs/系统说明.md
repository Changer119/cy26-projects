# 台风巴威实时监测大屏系统

## 系统概述

面向大屏展示的台风实时监测系统，自动获取 2026 年第 9 号台风「巴威」（BAVI，编号 202609）的最新信息，并以可视化大屏形式展示。**后端每 60 秒从数据源拉取一次最新数据，前端每 60 秒自动刷新展示。**

## 数据来源

- 浙江省水利厅 · 台风路径实时发布系统（`https://typhoon.slt.zj.gov.cn/Api`）
  - `TyphoonList/{year}`：年度台风列表（用于按英文名 BAVI 动态定位台风编号）
  - `TyphoonInfo/{tfid}`：台风详情（实况路径点、多机构预报路径、风圈半径、位置描述、趋势研判）

## 架构

```
浏览器大屏 (static/)  ──每60秒轮询──▶  FastAPI (/api/typhoon)
                                          │
                                     内存缓存 TyphoonCache
                                          ▲
                                   后台任务每60秒刷新
                                          │
                               浙江水利厅台风 API（数据源）
```

- `app/config.py`：全局配置（数据源、刷新周期、端口 8710）
- `app/models.py`：Pydantic 强类型模型（路径点、风圈、预报路径、大屏数据包）
- `app/fetcher.py`：数据抓取、解析与内存缓存，后台定时刷新（失败时保留上次数据）
- `app/server.py`：FastAPI 服务（`/api/typhoon`、`/api/health`、静态大屏页面）
- `static/`：大屏前端（Leaflet 地图 + 信息面板，深色主题）

## 大屏功能

- **地图中心**：实况路径（按强度分色）、五机构预报路径（中国/日本/美国/中国台湾/中国香港，虚线分色）、七级/十级/十二级风圈四象限扇形、旋转台风标记、图例
- **左侧面板**：实况信息（强度、风力、风速、气压、移向移速、经纬度、相对位置）、风圈半径、趋势研判
- **右侧面板**：中央气象台预报路径列表、距沿海城市实时距离（<200km 红色告警）、台风档案
- **顶部**：实时时钟、数据更新时间、下次刷新倒计时；**底部**：数据来源与系统状态

## 运行与调试

```bash
scripts/start.sh    # 启动（后台运行，自动 uv sync）
scripts/status.sh   # 查看状态 + 健康检查 + 最近日志
scripts/stop.sh     # 停止
```

- 大屏地址：<http://localhost:8710>
- 日志：`logs/app.log`（业务日志，轮转）、`logs/server.out`（进程输出）

## 关键设计

- **容错**：数据源刷新失败时保留上次缓存数据并记录日志，前端展示错误状态并自动重试
- **台风编号动态解析**：优先按年度列表中英文名 `BAVI` 查找，列表不可用时回退到配置编号 `202609`
- **强类型**：全部数据结构经 Pydantic 模型校验，风圈 `"280|180|220|180"` 解析为四象限（东北/东南/西南/西北）结构

# Phone2Computer 使用说明

## 推荐方式：华为 App（V2）

### 安装

1. 在 Mac 项目目录运行 `./scripts/start.sh`。
2. 用华为 P40 Pro 扫描 Mac 管理页二维码。
3. 首次使用点击“下载华为版 APK”，安装 `Phone2Computer-v2.apk`。
4. HarmonyOS 若提示“禁止安装未知应用”，仅为当前浏览器临时允许安装，安装后可关闭该权限。
5. 回到扫码页面，点击“在 App 中继续”，App 会自动获得本次 Mac 配对信息。

### 开启黑屏持续传输

首次安装后，在 App 中点击“设置黑屏后台传输权限”，并在华为系统中完成：

1. 允许 Phone2Computer 显示通知。
2. 进入“应用启动管理”，关闭自动管理。
3. 手动允许“自启动”“关联启动”“后台活动”。
4. 传输时不要在最近任务页面上滑强制关闭 App，也不要点击系统设置中的“强行停止”。

完成后，上传期间可以切换到其他页面或熄灭屏幕。通知栏会持续显示传输服务，并提供“暂停”按钮。

### 批量上传

1. 点击“批量选择照片和视频”。
2. 在华为系统文件选择器中多选文件；可以一次选择数百个。
3. 返回 App 后自动开始上传，默认同时传 6 个文件。
4. 网络短暂中断会自动重试；失败项可点击“重试失败”。
5. App 重启后点击“继续”，会从 Mac 已确认的分片位置恢复。

传输中的每个文件都会动态显示百分比和进度条，通知栏同时显示整体传输进度。

### APK 构建与有线调试

日常使用不需要 USB。开发调试时可通过统一脚本执行：

```bash
./scripts/mobile/setup.sh
./scripts/mobile/check.sh
./scripts/mobile/install.sh
./scripts/mobile/logcat.sh
```

APK 输出位置：

```text
dist/Phone2Computer-v2-debug.apk
```

手机客户端运行日志保存在 App 私有目录的 `files/logs/mobile.log`，开发调试日志同时可由 `mobile/logcat.sh` 查看。

## 兼容方式：手机浏览器（V1）

## 适用环境

- 发送端：华为 P40 Pro，HarmonyOS 4.2。
- 接收端：macOS。
- 手机和 Mac 必须连接同一个非访客 Wi-Fi，路由器不能开启设备隔离。

## 启动

在项目目录执行：

```bash
./scripts/start.sh
```

脚本会完成构建、启动本地服务并打开 Mac 管理页面。默认端口为 `18765`，默认接收目录为：

```text
~/Downloads/Phone2Computer
```

首次运行时，如果 macOS 询问是否允许接收入站连接，请选择“允许”。

## 从手机浏览器上传

1. 保持 Mac 管理页面打开。
2. 用华为手机相机扫描页面中的二维码。
3. 在手机页面点击“选择照片和视频”。
4. 从图库或文件管理中批量选择文件。
5. 上传期间保持浏览器页面在前台，并避免手机锁屏。
6. 页面显示“已完成”后，可在 Mac 接收目录中查看文件。

每次启动服务都会创建一个新的批次目录，例如：

```text
2026-07-18_161500_华为P40Pro
```

同名文件不会覆盖，系统会自动添加序号。

## 管理命令

所有运行和调试操作都通过项目脚本执行：

```bash
./scripts/status.sh
./scripts/stop.sh
./scripts/test.sh
./scripts/check.sh
```

如需修改接收目录：

```bash
PHONE2COMPUTER_OUTPUT_DIR="/目标目录" ./scripts/start.sh
```

如需修改端口：

```bash
PHONE2COMPUTER_PORT=18766 ./scripts/start.sh
```

## 故障排查

### 手机无法打开二维码地址

- 确认手机和 Mac 使用同一个 Wi-Fi。
- 不要使用访客 Wi-Fi。
- 检查 macOS 防火墙是否允许 Phone2Computer 接收入站连接。
- 企业网络可能开启客户端隔离，此时即使 Wi-Fi 名称相同也无法互访。

### 上传中断

- 保持手机浏览器在前台。
- 恢复 Wi-Fi 后，tus 客户端会自动重试。
- 自动重试耗尽后，点击“重试失败项”。

### 查看日志

运行日志统一保存在项目的 `logs/` 目录：

- `logs/app.log`：服务运行日志。
- `logs/build.log`：构建日志。
- `logs/test.log`：测试日志。
- `logs/check.log`：完整检查日志。

## 当前边界

- 只支持手机向 Mac 单向上传。
- 不保留手机端目录结构。
- 手机锁屏或浏览器退到后台后，HarmonyOS 可能暂停网页上传。
- 当前定位为可信局域网工具，不建议在公共 Wi-Fi 中使用。

# Phone2Computer 技术方案

## 目标

在同一 Wi-Fi 内，将华为 P40 Pro（HarmonyOS 4.2）的照片和视频批量上传到 macOS。手机无需安装应用，文件不经过云端。

## 已确认范围

- 手机通过浏览器扫码连接 Mac。
- 支持一次选择数百个照片和视频。
- 支持分片上传、暂停、重试和断点续传。
- 不保留手机原目录结构，Mac 按传输批次归档。
- 手机保持上传页面前台，不要求锁屏后台传输。
- 第一版只实现手机到 Mac 的单向传输。

## 架构

- Mac 接收端：Go 单进程 HTTP 服务，负责认证、tus 上传、文件落盘和静态资源服务。
- 手机与 Mac 页面：React + TypeScript，同一套前端根据路径展示上传页或管理页。
- 配对：服务启动时生成一次性高熵令牌，管理页使用二维码传递。
- 存储：文件先进入临时目录，完成后原子移动到本次传输批次目录。

## 安全边界

- 服务仅用于可信局域网，不主动开放公网入口。
- 所有上传接口要求 Bearer Token。
- 管理接口只允许本机回环地址访问。
- 服务端清理文件名并阻止路径穿越。
- 同名文件自动编号，默认不覆盖已有文件。
